# lyricsToImage

가사를 입력하면 이미지를 생성해주는 웹 애플리케이션입니다.

## 기능

- 사용자가 입력한 가사를 이미지로 변환
- 간단한 웹 UI 제공

## 사용 방법

1. 이 저장소를 클론합니다.
2. `index.html` 파일을 브라우저에서 엽니다.
3. 가사 및 아티스트명, 노래 제목을 입력하고 이미지를 생성하세요.

## 파일 구조

- `index.html` : 메인 HTML 파일
- `style.css` : 스타일시트
- `script.js` : 주요 기능이 구현된 자바스크립트 파일

## 기여

이 프로젝트는 AI를 이용하여 만들었습니다.
